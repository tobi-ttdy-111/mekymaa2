# Mekymaa

## Description

**Mekymaa** is a project designed to develop typing skills, allowing users to compete online with others. The platform offers various functionalities, such as creating groups, conversations, rooms, and more. Users can challenge each other to determine who is the best typist. Features include a rankings section to view the top players on the platform, the creation of public and private rooms, a ranked mode, global and private chats, 1v1 duels, and options for profile configuration and preferences. Mekymaa provides a diverse and versatile system in terms of features and aspects.

## Features

- **OAuth:** Account management and secure authentication.
- **Learning Modes:** Various modes to learn typing.
- **Real-Time Communication:** Global chats, private chats, and 1v1 duels.
- **Competitive:** Ranked mode and leaderboard system.
- **Customization:** Profile settings and preferences.
- **Responsive Design:** Modern user interface accessible from any device.

## Installation

Follow these steps to install and run Mekymaa locally:

1. Clone the repository:
    ```bash
    git clone https://github.com/your_username/Mekymaa.git
    cd Mekymaa
    ```

2. Install dependencies:
    ```bash
    npm install
    ```

3. Create a `.env` file in the project root, based on the `example.env` file:
    ```bash
    cp example.env .env
    ```

4. Configure the environment variables in the `.env` file as needed.

5. Start the server:
    ```bash
    npm start
    ```

## Usage

Mekymaa is designed to improve typing skills in a fun and competitive way. Users can:

- Learn typing through various practice modes.
- Participate in 1v1 duels and ranked competitions.
- Create and join public and private rooms.
- Chat in real-time with other users through global and private chats.
- Configure and customize their profile and preferences.
